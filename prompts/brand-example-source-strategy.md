# Brand Example Source Strategy

Use this when building prompt references from curated studio and agency portfolios.

## Goal

Create organized folders of real-world inspiration examples that help:
- build stronger product-visual prompts
- supply better presentation references to Replicate
- separate SaaS/product examples from premium-branding examples

## Category folders

Recommended structure:

```text
<working-folder>/brand-materials/examples/
  saas-product-specialists/
  premium-branding/
  strong-studios/
```

Each site should have its own folder:

```text
examples/saas-product-specialists/ramotion/
examples/premium-branding/pentagram/
examples/strong-studios/red-antler/
```

## What to capture

- homepage hero sections
- `/work`, `/projects`, or `/case-studies` pages
- standout browser or laptop product mockups
- isolated feature visuals
- strong background / framing examples

## What each category is best for

### SaaS & product specialists
Use for:
- browser framing
- dashboard crops
- startup-like polish
- feature-section visuals

### Premium branding
Use for:
- background systems
- editorial composition
- brand storytelling
- higher-end visual presentation

### Strong studios
Use for:
- broad exploration
- alternative layout ideas
- category-specific references like fintech, B2B, or startup launches

## Prompt-building rule

When using these examples, write down exactly what they control:
- crop
- whitespace
- background treatment
- browser/device framing
- depth and shadow level

Do **not** let them control:
- the actual app UI
- product copy
- buttons, cards, or labels
- fake product features
