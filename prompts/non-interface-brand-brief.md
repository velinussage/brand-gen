# Non-Interface Brand Brief

Use this before generating posters, pattern systems, sticker families, merch, or logo bumpers.

## Goal
Keep the artifact unmistakably branded even when there is little or no UI present.

## Midjourney-style lesson to borrow
- short, concrete prompts beat long adjective piles
- use reference images to carry style, composition, and finish
- keep the text prompt focused on **what** the thing is, not vague taste words
- if the prompt uses words like `premium`, `editorial`, or `system`, pair them with actual form nouns

Bad:
- premium editorial branded pattern system for an intelligent protocol

Better:
- square pattern board built from the exact brand mark anatomy: one horizontal cap, four vertical stems, copper square carriers, cream and espresso bands, one repeat tile, one border band, one emblem tile, quiet cream field

## Reference-image rule
For non-interface materials, do not rely on prompt text alone.

Use at least:
- 1 **brand truth** reference
- 1 **composition** reference
- 1 **application** or **motif** reference

Prefer **1 to 3 high-quality references total**, not a giant moodboard.  
Each reference should have a job:
- composition = framing and whitespace only
- motif = system mechanic only
- application = real-world landing surface only
- motion = reveal pacing only

Translate each reference into mechanics.
Do **not** borrow:
- the reference brand's logo
- the reference brand's typography
- the reference brand's copy
- the reference brand's literal symbol set

Whenever possible, crop reference images to the same aspect ratio as the target artifact.
Do not use full agency homepage screenshots when only one poster, tile, badge sheet, or motion frame is relevant.
If a reference looks like a whole portfolio wall, crop it or replace it.

## Concrete prompt recipe
Write prompts in this order:
1. exact mark anatomy
2. approved primitives
3. artifact count or system scope
4. composition structure
5. finish / material feel
6. negatives

Template:

```text
[exact mark anatomy] + [approved primitives] + [how many items / tiles / moves] + [layout or composition] + [finish] + [negative rules]
```

## Core rules
- show the stored logo / wordmark when the material is meant to be immediately recognizable
- if the logo is not shown, make the artifact clearly branded through at least 3 anchors: palette, mark geometry, approved motif, product truth, or brand name
- preserve the real logo/mark silhouette first
- preserve palette discipline before adding atmosphere
- preserve typography voice when text is composed deterministically
- derive graphic devices from the brand mark or product logic, not from random decorative trends
- pick **one system mechanic** before generating: banding, carrier repetition, cap-and-stem crop, routed line, mask wipe, etc.
- prefer one bold symbolic move over many unrelated embellishments
- prefer concrete nouns over abstract brand adjectives
- state the exact number of modules, stickers, tiles, or motion beats
- if the system could belong to any startup, it is not branded enough yet

## Ban these by default
- fake app names or repo names rendered as campaign copy
- invented product claims or startup boilerplate text
- fake chat bubbles, notifications, or generic dashboard chrome
- unrelated mascots, emojis, or decorative symbols
- generic neon/glass/scifi effects that erase the brand vibe
- abstract quarter-circle mosaics, generic knot symbols, or random badge silhouettes that are not clearly derived from the real mark

## Material differences
- campaign poster = fixed copy + one dominant brand field
- pattern system = repeatable identity logic, not a one-off illustration
- sticker family = small coherent set, each clearly descended from the mark
- merch poster = wearable / collectible / print-ready
- bumper / stinger = exact mark recognition + restrained reveal

## Quality reference heuristics
- The best composition references show **one dominant framing move**
- The best motif references show **how one primitive becomes a system**
- The best application references show **one real artifact family**, not a collage of everything the studio made
- The best motion references show **one reveal attitude** and a clean final hold

If a chosen reference does not make one of those jobs obvious in 2 seconds, it is the wrong ref for that role.

## Material-specific specificity targets
- campaign poster
  - one headline block
  - one logo carrier
  - one supporting band, line, or emblem move
- pattern system
  - one hero tile
  - one repeat tile
  - one border or band treatment
  - one emblem or carrier variation
- sticker family
  - 6 to 9 items max
  - 2 to 3 silhouette families max
  - one repeated line weight
  - one repeated carrier logic
- bumper / stinger
  - 4 to 6 seconds
  - one reveal idea
  - one final hold
  - no camera gymnastics unless the motion reference explicitly demands it
