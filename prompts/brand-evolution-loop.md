# Brand Evolution Loop

The goal is not one perfect output.

The goal is:
- remember what the brand is
- test how it can stretch
- ask for alignment when taste is subjective
- keep what works
- branch it into new materials

## Agent stance

The agent is allowed to:
- ask targeted questions when the direction is ambiguous
- show 2–3 idea tracks before generating
- check whether a result aligns with the user's taste
- revise the plan, not just the prompt

## Good loop

1. inspect brand memory
2. ideate 2–3 directions
3. ask 1–3 alignment questions
4. write one material plan
5. generate
6. critique
7. branch or refine
8. save what worked back into brand memory / plans

## What to preserve across time

- mark anatomy
- palette discipline
- approved primitives
- successful mechanics
- successful reference-role packs
- successful applications

## What can evolve

- composition
- cropping language
- poster attitude
- sticker family behavior
- motion pacing
- application style

## The test

Every cycle should answer:
- what did we learn about the brand?
- what direction got stronger?
- what should be repeated next time?
