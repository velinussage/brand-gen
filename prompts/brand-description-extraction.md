# Brand Description Extraction Prompt

Use this before generating any branded visual system outputs.

## Goal
Turn a product, codebase, website, or reference bundle into a structured brand profile that can drive:
- logo exploration
- landing hero composition
- browser illustrations
- product banners
- X / LinkedIn / OG cards
- X / LinkedIn feed posts
- feature animations

The extraction should produce:
- `brand-profile.json`
- `brand-profile.md`
- `brand-identity.json`
- `brand-identity.md`

## What to extract
- Brand / product name
- Short product description
- Audience
- Tone / adjectives
- Color candidates
- Semantic color roles / design tokens when available
- Typography cues
- Shape / UI cues
- Spacing, radius, shadow, and component cues when available
- Existing logo / icon assets
- Marketing / landing references
- External presentation references worth borrowing for composition
- Best product surfaces to screenshot

## Extraction questions
1. What is the product and who is it for?
2. What words best describe how it should feel?
3. What already exists that must stay recognizable?
4. What color, type, or interface cues are already present?
5. Which product screens best express the brand?
6. Which outputs are needed next: logo, banner, browser illustration, social card, or feature animation?
